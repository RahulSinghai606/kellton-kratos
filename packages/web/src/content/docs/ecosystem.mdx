A collection of community projects built on Kellton Kratos.

:::note
Want to add your Kellton Kratos related project to this list? Submit a PR.
:::

You can also check out [awesome-kratos](https://github.com/awesome-kratos/awesome-kratos) and [kratos.cafe](https://kratos.cafe), a community that aggregates the ecosystem and community.

| [kratos-google-antigravity-auth](https://github.com/shekohex/kratos-google-antigravity-auth)   | Google Antigravity OAuth Plugin, with support for Google Search, and more robust API handling  |

| [kratos-pty](https://github.com/shekohex/kratos-pty.git)                                       | Enables AI agents to run background processes in a PTY, send interactive input to them.        |
| [kratos-shell-strategy](https://github.com/JRedeker/kratos-shell-strategy)                     | Instructions for non-interactive shell commands - prevents hangs from TTY-dependent operations |
| [kratos-wakatime](https://github.com/angristan/kratos-wakatime)                                | Track Kellton Kratos usage with Wakatime                                                             |
| [kratos-md-table-formatter](https://github.com/franlol/kratos-md-table-formatter/tree/main)    | Clean up markdown tables produced by LLMs                                                      |
| [kratos-morph-fast-apply](https://github.com/JRedeker/kratos-morph-fast-apply)                 | 10x faster code editing with Morph Fast Apply API and lazy edit markers                        |
| [oh-my-kratos](https://github.com/code-yeongyu/oh-my-kratos)                                   | Background agents, pre-built LSP/AST/MCP tools, curated agents, Claude Code compatible         |
| [kratos-notificator](https://github.com/panta82/kratos-notificator)                            | Desktop notifications and sound alerts for Kellton Kratos sessions                                   |
| [kratos-notifier](https://github.com/mohak34/kratos-notifier)                                  | Desktop notifications and sound alerts for permission, completion, and error events            |
| [kratos-zellij-namer](https://github.com/24601/kratos-zellij-namer)                            | AI-powered automatic Zellij session naming based on Kellton Kratos context                           |
| [kratos-skillful](https://github.com/zenobi-us/kratos-skillful)                                | Allow Kellton Kratos agents to lazy load prompts on demand with skill discovery and injection        |

| [@plannotator/kratos](https://github.com/backnotprop/plannotator/tree/main/apps/kratos-plugin) | Interactive plan review with visual annotation and private/offline sharing                     |
| [@openspoon/subtask2](https://github.com/spoons-and-mirrors/subtask2)                              | Extend kratos /commands into a powerful orchestration system with granular flow control      |
