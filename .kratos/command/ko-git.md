Perform intelligent Git operations:

## Commands

### `commit` - Smart Commit
- Analyze staged changes
- Generate conventional commit message
- Include scope and breaking change detection
- Format: `type(scope): description`

### `branch` - Branch Management
- Create feature/bugfix/hotfix branches
- Follow naming conventions
- Set up tracking

### `pr` - Pull Request
- Generate PR description from commits
- Include testing checklist
- Link related issues

### `sync` - Repository Sync
- Fetch and rebase from upstream
- Resolve conflicts intelligently
- Preserve local work

### `changelog` - Generate Changelog
- Parse commit history
- Group by type (feat, fix, etc.)
- Generate release notes

Use Serena for code context in commit messages.
